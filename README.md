# deep-files-sync

Synchronizing text files in a folder on the local computer with Deep instance.


Installation:
```
npm install -g deep-file-sync
```
Usage:
```
deep-files-sync --url <deeplinks url> --space-id <space-id> --folder-path <path to folder> [--token <token>]
```
